# Restaurant-Analytics-Dashboard-Project-Overview

# Overview
This project is an interactive Restaurant Analytics Dashboard designed to provide deep insights into sales, orders, customers, menu performance, inventory, and employee data. The dashboard enables data-driven decision-making by visualizing key metrics and trends across multiple dimensions such as branches, regions, categories, and time.
